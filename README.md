# Quiz Game

A JavaScript exercise built while learning Full Stack Development.

## What it does

A simple quiz game where the computer randomly answers multiple choice questions across different categories.

- `getRandomQuestion()` — picks a random question from the list
- `getRandomComputerChoice()` — picks a random answer for the computer
- `getResults()` — checks if the computer's answer is correct

## How to run

```bash
node quiz-game.js
```

## What I practiced

- Working with arrays of objects
- Random selection with `Math.random()` and `Math.floor()`
- Basic game logic with functions
